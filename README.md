# 8Bid: Animated Desktop Companion 🐾

An interactive productivity timer designed to combat **office syndrome** by prompting users to "bid" (stretch) at regular intervals. 

## 🎯 Project Goal
The primary goal of 8Bid is to encourage healthy habits for desk-bound professionals. By utilizing the **Document Picture-in-Picture API**, 8Bid provides a persistent, floating animal companion that stays on top of other windows, reminding you to move, stretch, and stay hydrated without being intrusive.

## 🚀 Features
- **Floating Companion:** Use Document PiP to keep your animal friend visible while you work.
- **Smart Reminders:** Customizable intervals to remind you to stretch ("Bid").
- **Animated Sprites:** Choose between different animated companions (Black Cat, French Bulldog).
- **Interactive States:** Companions react to timer completion and have "nap time" states.
- **Custom Notifications:** Set your own reminder text.

## 🛠️ Tech Stack
- **Framework:** React 19
- **Language:** TypeScript
- **Build Tool:** Vite
- **Styling:** Vanilla CSS (Custom animations)
- **APIs:** Document Picture-in-Picture API, Web Audio API
